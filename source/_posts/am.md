## 怎么安装和使用
### 懒人一键安装
```
sh -c "$(curl https://raw.githubusercontent.com/Bcap03/AutoMux/master/script/install.sh)"
```

### 克隆使用
1. git clone https://www.github.com/Bcap03/AutoMux.git
2. pip install termcolor
3. cd AutoMux
4. python automux.py
